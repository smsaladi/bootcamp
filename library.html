Course Library

This page has a compilation of the course materials since its inception. As the course evolves so do does the subject matter, yet much still remains relevant reference material. State of computation is ever changing; some lessons may have outdated material.


Below, lessons are organized by topical area with the evening exercises generally excluded. If you'd like to work through the course yourself, consider using the materials in the order of the most recent edition.
